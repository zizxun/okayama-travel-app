# 岡山 2026/8/13-8/19 行程包

## 旅行設定

| 項目 | 內容 |
|---|---|
| 日期 | 2026/8/13 星期四 - 2026/8/19 星期三 |
| 進出 | 岡山桃太郎機場進出 |
| 航班 | 8/13 15:05 抵達；8/19 15:55 起飛 |
| 住宿 | 岡山站前大和 Roynet 飯店，6 晚；7 天都以岡山為基地 |
| 鐵路券 | JR-WEST 關西＆廣島地區鐵路周遊券，8/14-8/18 啟用 |
| 旅行節奏 | 舒服慢遊，上午排主景點，午後安排咖啡/商店/休息，夏天避免硬撐 |

## Pass 策略

- 啟用日：8/14-8/18 連續 5 天。
- 覆蓋：岡山到廣島、宮島口、姬路、倉敷、吉備津/吉備津彥相關 JR 移動，以及 JR 西日本宮島渡輪。
- 另付：岡山機場巴士、岡山市電/非 JR 巴士、宮島訪問稅 100 日圓、宮島纜車、餐飲、景點門票。
- 指定席建議：8/14 岡山-廣島、8/15 岡山-姬路、8/18 岡山-廣島都先劃位。回程可保留彈性，但熱門時段還是建議先劃。
- 大型行李：這趟都住岡山，不需要帶大行李跑新幹線；遠程日歸只帶小包和水。

## 8/13 星期四：抵達岡山

### 建議時間軸

| 時間 | 安排 |
|---|---|
| 15:05 | 抵達岡山桃太郎機場 |
| 15:55 或 16:55 | 搭機場巴士往岡山站西口。若入境很快，優先搭 15:55；若要領行李/補網卡，搭 16:55 也合理 |
| 16:25 或 17:25 | 抵達岡山站西口，步行到飯店寄放/入住 |
| 17:30-18:00 | 岡山站觀光案內所蓋紀念章，順便拿紙本地圖 |
| 18:00-20:00 | 永旺夢樂城岡山，晚餐 Myojinmaru 明神丸 |
| 20:00-21:00 | Yours 超市或站內採買：水、電解質飲料、早餐、水果 |

### 搭車步驟

1. 岡山機場到岡山站西口：機場巴士，自費 1,000 日圓，約 30 分鐘。
2. 岡山站西口到飯店：步行約 5-8 分鐘，或依行李量走地下/站前通道。
3. 飯店到永旺夢樂城：步行約 10-15 分鐘。

### 餐食候選

- 晚餐：Myojinmaru 明神丸 AEON MALL 岡山店，主打鰹魚稻草燒。
- 採買：Yours さんすて岡山店或站內便利商店。

### 太熱/下雨備案

- 直接飯店休息，永旺夢樂城改成只吃晚餐和採買。
- 紀念章若趕不上，移到 8/17 岡山市區日。

### 回飯店收尾

- 把 pass、護照、隔天要用的小包放好。
- 8/14 要很早出門，今晚只排輕鬆採買，不加夜景。

## 8/14 星期五：宮島日歸

目標是 11:00 前後在宮島大鳥居/嚴島神社一帶，配合你想看的滿潮時段。潮汐時間請出發前再查一次，以當日官方/港口潮汐表為準。

### 建議時間軸

| 時間 | 安排 |
|---|---|
| 07:15 | 飯店出門，岡山站買水和早餐 |
| 07:30-08:00 | 岡山站搭山陽新幹線往廣島，pass 覆蓋 |
| 08:15-09:00 | 廣島站轉 JR 山陽本線往宮島口 |
| 09:30-10:20 | 宮島口，搭 JR 西日本宮島渡輪到宮島，pass 覆蓋船資；入島稅 100 日圓另付 |
| 10:30-11:45 | 嚴島大鳥居、嚴島神社、海邊拍照 |
| 11:45-13:15 | 表參道商店街午餐：牡蠣、穴子飯、紅葉饅頭 |
| 13:15-15:30 | 宮島纜車與紅葉谷公園。若太熱，只到獅子岩展望台，不硬爬彌山山頂 |
| 15:30-17:00 | 咖啡/伴手禮/慢慢回碼頭 |
| 17:00-19:00 | 回廣島站，視體力在站內吃晚餐或直接回岡山 |

### 搭車步驟

1. 岡山 - 廣島：山陽新幹線，pass 覆蓋，建議劃指定席。
2. 廣島 - 宮島口：JR 山陽本線，pass 覆蓋。
3. 宮島口 - 宮島：JR 西日本宮島渡輪，pass 覆蓋船資；訪問稅 100 日圓另付。
4. 宮島纜車：pass 不覆蓋，自費，出發前確認營運與風雨狀況。

### 餐食候選

- 牡蠣：焼がきのはやし、牡蠣屋、表參道沿線烤牡蠣攤。
- 甜點：紅葉堂、藤い屋、やまだ屋的紅葉饅頭。
- 若排隊太長：先買小吃，正餐回廣島站吃。

### 太熱/下雨備案

- 取消纜車，改成嚴島神社、表參道商店街、咖啡店和伴手禮。
- 若渡輪因天候變動，依 JR 宮島渡輪公告調整，宮島日可與 8/18 廣島市區交換。

### 回飯店收尾

- 回岡山後只在站內或飯店附近簡單吃。
- 隔天姬路日要走樓梯，準備厚襪子和輕便鞋。

## 8/15 星期六：姬路 + 吉備津彥神社/吉備津神社

這天原始安排偏滿，而且 8 月姬路城內沒有空調、樓梯陡。標準版可行，但如果身體覺得熱，請採用降載版。

### 標準版時間軸

| 時間 | 安排 |
|---|---|
| 08:00 | 岡山站出發，山陽新幹線往姬路，pass 覆蓋 |
| 08:30-09:00 | 姬路站抵達，步行或公車到姬路城 |
| 09:00-11:30 | 姬路城。先進天守，之後再看外圍 |
| 11:30-12:30 | 姬路城周邊午餐或姬路站午餐 |
| 12:30-13:30 | 返回岡山 |
| 14:00-15:00 | JR 桃太郎線到備前一宮站，參拜吉備津彥神社 |
| 15:00-16:30 | JR 或計程車到吉備津神社，走廊、本殿、鳴釜神事外觀 |
| 16:30-17:30 | 回岡山站 |
| 晚上 | 岡山站周邊晚餐，早點休息 |

### 降載版

- 只做姬路城 + 好古園，下午回岡山休息。
- 吉備津彥神社/吉備津神社移到 8/17 上午或 8/18 若不去廣島時備用。

### 搭車步驟

1. 岡山 - 姬路：山陽新幹線，pass 覆蓋，建議劃指定席。
2. 姬路站 - 姬路城：步行約 20 分，或北口公車到大手門前。
3. 岡山 - 備前一宮/吉備津：JR 桃太郎線，pass 覆蓋。

### 太熱/下雨備案

- 姬路城內樓梯陡又悶熱，若排隊或高溫警報，改拍外觀 + 好古園。
- 吉備津兩社之間不建議夏天硬走山路，可用 JR 班次或短程計程車。

### 回飯店收尾

- 今天腳會累，晚上避免再排商場。
- 檢查 8/16 倉敷是否想穿好走的鞋，阿智神社有坡。

## 8/16 星期日：倉敷慢遊

動線修正：倉敷美觀地區在倉敷站南口方向，Outlet 在北口方向。建議先南口老街，下午回站穿越到北口 Outlet。

### 建議時間軸

| 時間 | 安排 |
|---|---|
| 09:00 | 岡山站搭 JR 山陽本線到倉敷，pass 覆蓋 |
| 09:20-09:40 | 倉敷站南口，慢慢走到美觀地區 |
| 09:45-10:30 | 阿智神社。先爬坡，趁早上比較不熱 |
| 10:30-12:30 | 倉敷美觀地區、白壁街道、運河、雜貨店 |
| 12:30-13:30 | 午餐：岩むら或美觀地區附近和食 |
| 13:30-15:30 | 大原美術館或咖啡休息；不想進館就繼續逛商店 |
| 15:30-17:30 | 回倉敷站，北口 Mitsui Outlet Park 倉敷 |
| 17:30-19:00 | 倉敷晚餐或回岡山站吃 |

### 餐食候選

- 正餐：御食事処 岩むら。若滿席，改美觀地區附近和食/蕎麥/咖啡餐。
- 小吃：可樂餅、炸肉餅、倉敷布丁、白桃甜點。
- 紅葉饅頭：不是倉敷代表點心，建議 8/14 宮島或 8/18 廣島補買。

### 太熱/下雨備案

- 阿智神社取消，直接美觀地區 + 大原美術館 + 咖啡。
- Outlet 變成避暑主力，晚餐回岡山站。

### 回飯店收尾

- 整理伴手禮，避免最後一天才買太多。
- 8/17 是岡山市區日，可以稍微晚起。

## 8/17 星期一：岡山城 + 後樂園 + 表町商店街

後樂園 3/20-9/30 通常 7:30-18:00，岡山城通常 9:00-17:30。夏天建議早上先後樂園，避開午後烈日。

### 建議時間軸

| 時間 | 安排 |
|---|---|
| 08:00 | 飯店出門，岡山站搭市電或公車到後樂園/城下 |
| 08:30-10:00 | 岡山後樂園，先走主要庭園和拍岡山城借景 |
| 10:00-11:30 | 岡山城天守 |
| 11:30-12:30 | 城下或後樂園周邊午餐/茶屋 |
| 12:30-14:30 | 表町商店街、天滿屋、咖啡休息 |
| 14:30-16:00 | 回飯店午休或補岡山站紀念章 |
| 晚上 | 壽喜燒晚餐，需事前預約 |

### 壽喜燒預約任務

- 優先條件：岡山站或表町附近、可線上預約、晚餐 18:00-19:00、可接受外國旅客。
- 搜尋關鍵字：`岡山駅 すき焼き 予約`、`岡山市 すき焼き 個室`、`Okayama sukiyaki reservation`。
- 若未訂到：改吃岡山站/永旺夢樂城餐廳，或把壽喜燒改到廣島/姬路日。

### 太熱/下雨備案

- 後樂園只走 60-75 分鐘，取消深度繞行。
- 表町商店街和天滿屋做避暑主力。
- 若下大雨，岡山城 + 表町 + 天滿屋，後樂園改成短暫拍照。

### 回飯店收尾

- 8/18 又是廣島日歸，晚餐不要太晚。
- 把 pass 放在小包固定位置，最後一天已不再使用 pass。

## 8/18 星期二：廣島市區日歸

8/14 已去宮島，這天聚焦廣島市區。節奏建議沉穩一些，和平紀念資料館需要心理和時間餘裕。

### 建議時間軸

| 時間 | 安排 |
|---|---|
| 08:30 | 岡山站出發，山陽新幹線往廣島，pass 覆蓋 |
| 09:15-10:00 | 廣島站，轉市電/巴士往和平紀念公園，自費或現場確認 pass 是否可用於 JR 巴士路線 |
| 10:00-12:30 | 原爆圓頂館、和平紀念公園、和平紀念資料館 |
| 12:30-13:30 | 本通/紙屋町午餐 |
| 13:30-15:30 | 本通商店街、紙屋町、廣島城外觀或縮景園二選一 |
| 15:30-17:30 | 咖啡/伴手禮，廣島站補買紅葉饅頭 |
| 18:00-20:00 | 新幹線回岡山，站內或飯店附近晚餐 |

### 餐食候選

- 廣島燒：本通/八丁堀/廣島站 ekie 一帶。
- 伴手禮：紅葉饅頭、生もみじ、檸檬系點心。
- 如果 8/14 沒吃夠牡蠣，這天在廣島站補。

### 太熱/下雨備案

- 和平紀念資料館 + 本通商店街 + 廣島站，取消戶外延伸。
- 若資料館排隊過長，先走公園與原爆圓頂館，下午再回資料館。

### 回飯店收尾

- 今晚整理行李。
- 檢查 8/19 機場巴士、護照、票券、液體和行李重量。
